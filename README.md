## 🚀 Kurulum ve Güncelleme

### Yerel Geliştirme
```bash
# Repoyu klonla
git clone https://github.com/hakanerbasss/tchat.git

# Dizine gir
cd tchat

# Tarayıcıda aç
# index.html dosyasını tarayıcıda açın
```

### GitHub Pages Güncelleme
1. Dosyaları düzenle
2. GitHub'a push et
3. GitHub Pages otomatik deploy eder

```bash
git add .
git commit -m "Güncelleme açıklaması"
git push origin main
```
